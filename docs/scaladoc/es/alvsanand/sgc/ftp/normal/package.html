  1. object FTPSgcConnectorFactory extends SgcConnectorFactory[FTPSlot, FTPParameters]

    This Factory creates instances of es.alvsanand.sgc.core.connector.SgcConnector for integrating with a FTP server.

    This Factory creates instances of es.alvsanand.sgc.core.connector.SgcConnector for integrating with a FTP server. It list and fetch all the files that are in a configured directory.

    Note: every file will be used as a slot.

    It has these features:

    • The FTP client will authenticate using the credentials.
    • If the proxy parameters are set, the FTP client will use a proxy instead of a direct connection.
