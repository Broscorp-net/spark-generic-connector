  1. object FTPSSgcConnectorFactory extends SgcConnectorFactory[FTPSlot, FTPSParameters]

    This Factory creates instances of es.alvsanand.sgc.core.connector.SgcConnector for integrating with a FTPS server.

    This Factory creates instances of es.alvsanand.sgc.core.connector.SgcConnector for integrating with a FTPS server. It list and fetch all the files that are in a configured directory.

    Note: every file will be used as a slot.

    It has these features:

    • The FTP client will authenticate using the credentials.
    • If the keystore is set, the FTPS client set NeedClientAuth to true. That means the client must use a a certificate to create the SSL connection and the server must validate the client certificate.
    • If truststore is set, the FTPS client will check that the server certificate is valid using that truststore. That means that if the sever certificate is not in the truststore the connection will fail.

    This Factory creates instances of es.alvsanand.sgc.core.connector.SgcConnector for integrating with a SFTP server. It list and fetch all the files that are in a configured directory.

    Note: every file will be used as a slot.

    It has these features:

    • If the keystore is set, the SFTP client will use the private key to authenticate instead of the password.
