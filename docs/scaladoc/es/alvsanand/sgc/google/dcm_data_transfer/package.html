  2. object DataTransferSgcConnectorFactory extends SgcConnectorFactory[DataTransferSlot, DataTransferParameters]

    This Factory creates a instances of es.alvsanand.sgc.core.connector.SgcConnector for integrating with DoubleClick Data Transfer.

    This Factory creates a instances of es.alvsanand.sgc.core.connector.SgcConnector for integrating with DoubleClick Data Transfer. It list and fetch all the files that are in a configured bucket.

    Note: every file will be used as a slot.

    It has these features:

    • The CloudStorage client will authenticate using the credentials zip. See es.alvsanand.sgc.google.GoogleHelper to know more about the credentials zip file.
    • The Google authentication method used by this implementation is Authorization code flow using a GooglePromptReceiver. See Authorization Code Flow for more info.
    • The SgcConnector is able to filter the type of Data Transfer files.
